# Юферова Анна Владимировна АТ-01

## Книжный интернет-магазин
Цель проекта: создать бэкенд интернет-магазина, специализирующегося на продаже книг различных жанров и направлений. Он будет обеспечивать необходимый функционал для удобного использования сервиса как клиентами, так и администраторами.

### Магазин для обычного пользователя
При входе на сайт пользователь попадает на главную страницу. На ней будущий покупатель может воспользоваться поисковой строкой или уже готовыми разделами товаров, чтобы отыскать интересующие его книги. В ленте будут отображаться карточки товара с обложкой книги, названием и краткими сведениями. Нажав на нужную книгу, пользователь попадает на её страницу, где есть предисловие и также дополнительная информация. 
В карточке книги, которая находится на общей странице, и на странице товара присутствует кнопка «Купить», нажав на которую покупателя попросят зарегистрироваться на сайте для совершения покупки. После этого пользователь может перейти в «Корзину», где есть возможность удалить товар или продолжить оформление покупки, нажав на кнопку «Перейти к оформлению». 
На странице «Оформление заказа» покупатель проверяет свои данные и привязывает карту.
Также в карточке товара, пользователь может забронировать интересующую его книгу, если ее нет в наличии. Это можно сделать, нажав на кнопку «Забронировать книгу».

### Магазин для администрации веб-приложения
Администратор может добавить новые книги на общей странице товаров или менять на странице товара наличие книги. Также будет присутствовать функция добавления раздела поиска. Зайдя в свой профиль, администратор может увидеть новых зарегистрированных пользователей магазина.

### Функциональность приложения
1.	Создание профиля: пользователи могут зарегистрироваться на сайте и создать свой профиль.
2.	Просмотр товара: пользователи имеют возможность просматривать товары в виде общей странице со всеми товарами.
3.	Поиск товара: пользователи смогут найти интересующие их товары поиском по категориям, ценам, рейтингу или издателям.
4.	Корзина: пользователи могут добавлять, удалять и оформлять заказ в корзине.
5.	Рейтинг товаров: покупатели смогут оставлять комментарии и оценивать товар.
6.	Бронирование товара: пользователи могут забронировать товар, при его отсутствии.
7.	Администрирование сайта: администраторы могут управлять товарами, категориями и пользователями.
